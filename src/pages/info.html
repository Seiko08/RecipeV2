📌 Projet Scolaire

Ce projet a été réalisé dans le cadre du cours de Programmation et Développement Web. Il constitue l’épreuve finale de ce module et démontre les compétences acquises tout en respectant les consignes suivantes :

📝 Contraintes du projet

  • 🗄️ Interagir avec une base de données IndexedDB.
  • 📦 Utiliser une librairie externe installée via NPM.
  • 🎯 Implémenter au moins 3 EventListeners déclenchant des actions.
  • 🌍 Utiliser des requêtes AJAX pour consommer une API REST.
  • 🎭 Ajouter au moins 3 animations JavaScript.
  • ⚡ Application 100% dynamique sans rechargement de page.
  • 🔗 Versionner le projet sur GitHub.
  • 📝 Effectuer des commits réguliers à chaque avancée.

📧 Validation du sujet

Le sujet est libre mais doit être validé par le chargé de cours. Cette validation doit être effectuée par email afin de conserver une trace écrite.
